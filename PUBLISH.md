# Publish Guide (GitHub Pages • Website • LinkedIn)

This kit is already in “drop‑in” format:
- one folder
- one `START_HERE.html`
- three levels
- signed proof packs

---

## 1) Publish on GitHub (repo + GitHub Pages)

### A) Create the repo
1. Create a new public GitHub repository.
2. Upload everything in this folder (keep the folder structure).

### B) Turn on GitHub Pages
1. Go to **Settings → Pages**
2. Source: **Deploy from a branch**
3. Branch: **main** / root
4. Save

### C) Share the link
Your entrypoint is:

`START_HERE.html`

---

## 2) Publish on Vercel / Netlify / Cloudflare Pages

Any static host works.

- No build command
- Output / publish directory: the folder root

---

## 3) Put it on your website

Recommended:
- Host the interactive page (shareable URL)
- Offer the proof packs as downloads

Suggested CTA buttons:
- **Start the Challenge** → `START_HERE.html`
- **Download a Proof Pack** → `sample_packs/.../*.zip`
